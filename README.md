# 🧪 REACTRACK

*REACTRACK* is a web application built with *Streamlit* for intelligent management of laboratory reagents.

It enables laboratories to:

- Track reagent usage
- Identify critical stock levels
- Analyze reagent consumption per user
- Collect user feedback through an integrated feedback system
- Anticipate future needs and optimize stock management

The objective of REACTRACK is to improve *scientific organization and strategic resource management in laboratories* through simple and interactive data visualization.


# 💡 Key Features

## Interactive Dashboard
- Visualization of critical and sufficient reagent stocks
- Identification of the most consumed reagents
- Clear overview of laboratory reagent usage

## Stock Management
- Register new reagents
- Record reagent usage
- Monitor stock evolution

## Data Analysis
- Visualization of stock trends over time
- Heatmap showing *User × Reagent consumption*

## User Feedback
- Integrated page allowing users to submit feedback
- Comment field with optional email for follow-up
- Feedback used to improve the application

## Security
- Password-protected access
- Secure user session management with logout option
